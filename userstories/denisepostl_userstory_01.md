# User story add router to system.

Add a router to the system.

## Actors

* Person (Role: Admin)

## Input

Building, Floor, Room number, Router numbers

## Internal state change

The router can be replaced.

## Output 

Acknowledgement

## Errors

* insufficient rights
* nonexistent building
* nonexistent floor
* nonexistent room 
* already existent router number
* defect router
