# User story add switch to system.

Add a switch to the system.

## Actors

* Person (Role: Admin)

## Input

Building, Floor, Room number, Switch numbers

## Internal state change

The switch can be replaced.

## Output 

Acknowledgement

## Errors

* insufficient rights
* nonexistent building
* nonexistent floor
* nonexistent room 
* already existent switch number
* defect switch
